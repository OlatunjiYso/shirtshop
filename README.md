###### Shirtshop is an e-commerce site that allows users to do the following :
- view shirts by categories
- search for shirts
- view the details of a particular shirt
- signup and login
- add shirts to shopping cart
- checkout and make payments

###### Technology Stack
- Node
- Express
- Mysql
- React.js
